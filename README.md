# CosmoQuiz - Space Academy 🚀

Welcome to **CosmoQuiz**, an interactive kindergarten-level solar system quiz application completely **vibecoded** into existence! 🌌✨

Starting from a conceptual design generated via Google Stitch ("Neon Observatory" / Space Night theme), the entire application was built using AI assistance (vibecoding) to bring the static designs to life with React, React Router, and Tailwind CSS.

## Features 🪐
- **Space Night Design System**: No 1px borders here! Only glassmorphism, nebula backgrounds, gradients, and soft floating interactive elements.
- **5 Interactive Questions**: Educational and playful questions about Earth, the Sun, Mars, the Moon, and Saturn.
- **Fun Feedback & Gamification**:
  - Orbit progress indicators displaying current sequence.
  - "Stellar Job!" robot modals upon correct answers, complete with **confetti bursts** 🎉
  - "Oh no, Space Cadet!" feedback upon incorrect answers.
- **Mission Complete Dashboard**: See your final score out of 5 stars and easily replay your mission!

## Tech Stack 🛠️
- **React 19**
- **Vite**
- **Tailwind CSS v3** (with custom Space Night theme tokens)
- **React Router v6**
- **React Confetti** & **React Use** (for joyful success metrics)

## How to Run locally 💻
```bash
# Install dependencies
npm install

# Run the development server
npm run dev
```
